This page template is a quick way to start any new project, especially great for dev-related content.

On this page, you see examples of:

  • Twitter Bootstrap components - make sure you're familiar with them.
  • Compact Kyruus-themed navbar - uses the Kyruus color scheme and uses minimal space.
  • Sticky Footer - notice how the footer stays at bottom of the page?
  • Font Awesome - vector icons that look good at every size.
